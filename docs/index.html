FLOORGUIDEDON project by Takayuki Ako

FLOORGUIDEDONの目指すところ

　博物館・美術館、図書館、ギャラリーなどのフロアガイドを、誰もが自由に創造、利用できると楽しそうです。公式フロアガイドの他にも、様々なフロアガイドがあるといいですよね。

　しかし、実際作ろうと思うと、作品情報や館内地図など揃えないといけない素材が多いことに気づきます。それがいま、WikipediaやOpenStreetMapなどを活用することで割と簡単に実現できそうです。

　FLOORGUIDEDONでは、あなたのフロアガイドづくりをお手伝いしたいと考えています。
